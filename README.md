# LangTrainer — тренажёр английского языка с интервальными повторениями

Веб-приложение для изучения английских слов, использующее алгоритм интервальных повторений, умную проверку ответов (лемматизация + синонимы) и ограничение попыток в день. Проект выполнен в рамках учебного задания.

## Основные возможности

- Регистрация и аутентификация пользователей.
- Выбор урока из базы.
- Интервальные повторения: каждое слово имеет уровень (0–5) и дату следующего повторения. Правильные ответы увеличивают интервал, неправильные сбрасывают уровень.
- Умная проверка ответов: учитываются разные формы слова (падежи, числа) и синонимы.
- Лимит 5 попыток в день (каждая проверка считается попыткой, независимо от правильности).
- Статистика: количество выученных слов, общая успешность, попытки сегодня.
- CRUD для слов и уроков (только для администратора).
- API для получения списка слов (JSON).
- Адаптивный интерфейс на Bootstrap 5.
- Встроенное аудио (синтез речи Web Speech API).

## Технологии

- Python 3.14
- Django 6.0
- SQLite (для разработки)
- Bootstrap 5
- Django REST Framework
- pymorphy2, ruwordnet (для умной проверки)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/LapyginDmitriy/langtrainer.git
cd langtrainer

# Создать и активировать виртуальное окружение
python -m venv venv
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Установить зависимости
pip install django djangorestframework pillow pymorphy2 ruwordnet setuptools pylint pylint-django

# Применить миграции
python manage.py migrate

# Создать суперпользователя
python manage.py createsuperuser

# Запустить сервер
python manage.py runserver

Откройте в браузере http://127.0.0.1:8000.

##Начальная настройка
Зайдите в админку (/admin) и создайте урок (Lesson).

Добавьте несколько слов (Word) в этот урок.

Зарегистрируйтесь как обычный пользователь. При регистрации автоматически создаются записи UserWord для всех существующих слов.

Выберите урок на странице "Выбрать урок" и начинайте повторение.
